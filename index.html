Web Scrapers Sample Code

Installation, Usage and Source Code Download

Environment Setup

  • Ruby 1.9.2 (Linux Family | Windows)
  • Ruby Gems
    • mechanize 
      gem install mechanize
    • sequel 
      gem install sequel
    • mysql (the path for with-mysql-config depends on your installation environment) 
      gem install mysql -- --with-mysql-config=/usr/bin/mysql_config
  • MySQL Database (required if you are going to use Database instead of CSV file)
    • Download Installer
    • Download Admin GUI (use Sequel Pro if you use Mac OSX)
    • After installation, create a database web_scrapers for the program to store data

Step by Step tutorial, use openrice.com as an example

  1. Grab available districts
  2. Grab available cuisines
  3. Submit search by district and cuisine
  4. Scrape restaurant info from search result
  5. Parse pagination links and scrape all result pages
  6. Final: Save as CSV | Store into Database

How to run the scrapers

command to scrape restaurant data and store as .CSV file

$ ruby final_open_rice_web_scraper_csv.rb

command to scrape restaurant data and store into MySQL DB

$ ruby final_open_rice_web_scraper_mysql.rb
